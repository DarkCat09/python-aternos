Uploading files and FTP access

Unfortunately, Aternos doesn't offer access via FTP or the ability to upload custom mods/plugins. There are many reasons for this decision but it's mainly for security reasons.

Plugins or mods uploaded to your server run on our hardware. They might contain suspicious software code which is then executed in our environment. This might be a bot or a type of virus.
